Política de Privacidade

Última atualização: abril de 2026

Este site, floriano.des.br, é o portfólio pessoal e profissional de Floriano Silva, Product Designer com foco em growth, CRO e analytics. Esta política explica de forma transparente quais dados são coletados, como são usados e quais são os seus direitos.

1. Dados coletados

Este site não coleta dados pessoais diretamente. Não há formulários de cadastro, login ou área restrita. Os únicos dados processados são:

  • Dados de navegação anônimos — coletados via Google Tag Manager / Google Analytics (endereço IP anonimizado, páginas visitadas, tempo de sessão, dispositivo e navegador).
  • Dados de contato — quando você envia um e-mail para contato@floriano.des.br ou entra em contato pelo WhatsApp, os dados informados ficam armazenados apenas no provedor de e-mail/mensagem utilizado.

2. Finalidade do uso dos dados

Os dados de navegação são usados exclusivamente para entender o desempenho do site (quais páginas são mais acessadas, origem do tráfego) e melhorar a experiência do usuário. Nenhum dado é vendido ou compartilhado com terceiros para fins comerciais.

3. Google Tag Manager e Analytics

Este site utiliza o Google Tag Manager (GTM-K738G4S2) para gerenciar scripts de análise. O Google Analytics coleta dados de forma agregada e anonimizada. Você pode optar por não ser rastreado instalando a extensão oficial: Google Analytics Opt-out Browser Add-on .

4. Cookies

O site pode utilizar cookies funcionais e de análise (originados do Google Analytics). Você pode desativar cookies nas configurações do seu navegador a qualquer momento, sem prejuízo ao acesso ao conteúdo.

5. Links externos

Este site contém links para LinkedIn, WhatsApp e outros serviços externos. Ao acessá-los, você estará sujeito às políticas de privacidade de cada plataforma. Não nos responsabilizamos pelo conteúdo ou práticas desses serviços.

6. Seus direitos (LGPD)

Nos termos da Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento de seus dados;
  • Solicitar acesso, correção ou exclusão dos dados;
  • Revogar consentimento a qualquer momento.

Para exercer esses direitos, entre em contato pelo e-mail: contato@floriano.des.br.

7. Alterações nesta política

Esta política pode ser atualizada a qualquer momento. A data de última atualização sempre estará indicada no topo desta página.
